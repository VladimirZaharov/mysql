Моя работа посвящена сайту кулинарного направления "поварёнок"(povarenok.ru). Хранение рецептов от пользователей.
Рецепты сосредоточены по категориям блюда, национальных кухонь, назначению того или иного рецепта.
В базе представляется полное описание рецептов с описанием необходимых ингредиентов, приложением медиафайлов, комментариями от других пользователей, возможностью подсчета лайков.
