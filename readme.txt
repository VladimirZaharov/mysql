Моя работа посвящена сайту кулинарного направления "поварёнок"(povarenok.ru). Хранение рецептов от пользователей.
Рецепты сгруппированы по категориям блюда, национальных кухонь, назначению того или иного рецепта.
В базе представляется полное описание рецептов с описанием необходимых ингредиентов, приложением медиафайлов(фотографии от пользователей, видео), комментариями от других пользователей, возможностью подсчета лайков для последующей выборки наиболее популярного рецепта.
